(about 15 minutes) Objective: Make and run a custom Web Component.

What Are Custom Components?

The video tag is a built in or pre-defined Web Component. It is possible to create your own Web Components. However, the standards for native Web Components isn't done yet and not all browsers support current ideas for custom Web Components. For the best compatibility across browsers a shim library has to be used. We will get into the specifics of this in a future lesson.

For now, we will create a native Web Component that will work in at least Chrome.
