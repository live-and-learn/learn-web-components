Introducing the HTML <video> tag.

With and since the introduction of HTML5 we actually already have some 'Web Components' in the form of some new HTML tags that are more complex than tags like the <h1> tag and other tags that have been around for many, many years.

The <video> tag looks simple like other HTML tags, but you will notice that it has a set of video controls. How can a simple HTML tag have complex features like that?

When you use the <video> tag the web browser knows the additional controls to add to make the tag look and work correctly. And there are actually other HTML elements 'hidden' within the <video> tag. This is one of 4 ideas that groups are trying to standardizeas part of the Web Components standards. This 'hidden' area of a Web Component is known as the "Shadow DOM" and is like the DOM representation of a web page, but just for a specific control like the <video> tag.

Now let's take a look at the <video> tag.


        <video src="./welcome.mov" type="video/mov" controls></video>
